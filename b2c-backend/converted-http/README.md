# Converted Http Layer

This folder contains JavaScript counterparts for the Laravel `Http` folder.

- Source: `../Http`
- Output target: `b2c-backend/converted-http`
- Generated by: `npm run generate:http-js`

Notes:
- Every PHP file under `Http` gets a JS counterpart here.
- Most files are generated as Express-oriented placeholders so the folder structure is preserved.
- `Controllers/Admin/AdController.js` is implemented as a real Express controller and can be wired into routes.
